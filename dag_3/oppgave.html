Oppgave dag 3

Forord

I dag skal vi fortsette med webshop, og introdusere noen nye konsepter som vil gjøre det lettere å holde orden på elementer og verdier, samt manipulering av DOM (Document Object Model).

Webshoppen skal kunne..

  • Vise visuelt hva som er i handlekurven
  • Slette enkeltelementer fra handlekurven
  • Dynamisk oppdatere handlekurven
  • Vise totalsummen av elementer i handlekurven

Enkelt forklart skal dere lage en mer interaktiv og intuitiv handlekurv enn tidligere, og jobbe med JavaScript for å oppnå dette.

De tekniske målene for dagen er:

  • HTML og CSS for å legge til visning av en handlekurv nederst på siden
  • Bruke Javascript for funksjonalitet til å legge til og slette elementer i handlekurven
  • Innføre JSON objekter for hvert av elementene man kan legge til i handlekurv

Oppgaver

I dag skal vi starte med å lage designet til handlekurven, samt å legge til noe Javascript for å vise hvilke elementer man kan har lagt til som varer i handlekurven.

1. Oppsett av handlekurven

Vi starter først med å åpne HTML-filen vi lagde i går. Inne i denne kan du starte med å lime inn følgende:

                    <div class="items">
                        <p>Handlekurv:</p>
                        <ul class="list-group" id="handlekurv">
                            <!-- Her skal elementene (list items) vises.
                            List items lages med:
                                <li>"navn på element"</li>
                                -->
                        </ul>
                    </div>

Dette er da HTML-en til handlekurven vi skal jobbe med. Vi skriver som nevnt i forelesning, kode som skal vise containeren til handlekurven.

Oppgave:

Legg til noen list items i handlekurven.

Prøv å bruke CSS til å style handlekurven slik du vil at den skal se ut.

2. DOM-manipulasjon

Neste steg er å begynne med DOM-manipulasjon. Det vil si endring av HTML-dokumentet, ofte ved bruk av JavaScript.

Her er et eksempel:

Denne funksjonaliteten oppnår man med denne koden:

                <script>
                    function test() {
                        var element = document.getElementById('test');
                        element.innerHTML = 'Du la til denne teksten';

                    }
                </script>
                <ul id="eksempel">
                    <button onclick="test()">
                        Trykk på meg
                    </button>
                </ul>
                <p id="test">
                </p>

Videre kan man legge til betingelser (en if/else-test) som gjør at man kan endre oppførsel avhengig av betingelser.

Les gjerne mer om "if/else-test" her (dere trenger ikke lære om if/else ennå, men ta gjerne en titt hvis det virker interessant).


Dette kan du få til ved å endre JavaScript-koden til følgende:

                    var count = false;

                    function test() {
                        var element = document.getElementById('test')
                        element.innerHTML = 'Du la til denne teksten';
                        var btn = document.getElementById('btn');
                        if (count) {
                            element.style.display = 'none';
                            count = false;
                            btn.innerText = "Trykk på meg";
                        }
                        else {
                            element.style.display = 'block';
                            count = true;
                            btn.innerText = "Trykk på meg igjen"
                        }
                    }

I tillegg må du gi knappen en ID:

                     <ul id="eksempel">
                        <button id="btn" onclick="test()">
                            Trykk på meg
                        </button>
                    </ul>
                    <p id="test">
                    </p>

Nå ser vi at man kan bruke JavaScript til å manipulere DOM (HTML) på nettsiden vår.

Oppgave:

Legg JavaScript-koden i JavaScript filen din (script.js) og se til at den fungerer på samme måte der.

Når du har gjort dette, kan du legge til funksjonalitet for å endre handlekurven basert på elementer lagt til i den.

Start med å endre teksten i handlekurven til å vise det siste elementet lagt til. Eksempel "Datamaskin" eller "Mobil"

Tips: Se på funksjonen "function test()" i eksempelet over, og lag en funksjon i ditt script.js-dokument som ligner.

Om du sitter fast, spør en av foreleserne. Du kan også ta en titt på hintet eller løsningsforslaget under:

Legg merke til document.getElementById og innerHTML

Se om du kan få innerHTML i handlekurven til å vise det som ble lagt til.

Du må bruke item som er brukt i addItem-funksjonen i script.js-filen.

                        function addItem(item) {
                            items.push(item);
                            cartAdd(item);
                        }

                        function cartAdd(item){
                            var kurv = document.getElementById('handlekurv');
                            kurv.innerHTML = item;
                        }
3. Legge til HTML i handlekurv

Nå skal vi vise hvilke produkter som ligger i handlekurven vår. Dette oppnår vi ved bruk av DOM-manipulasjon og å legge til HTML i form av list-items i listen vår.

Eksempelet under viser hvordan man kan legge til list-items i en liste. Det er i tillegg lagt til funksjonalitet for å velge tekst som skal legges til i listen.

Dette vil kun bli brukt som eksempel på hva som er mulig, og det er ikke nødvendig å forstå all JavaScript-koden.

Legg spesielt merke til outputTest(text)-funksjonen. Den ligner veldig på den vi kommer til å bruke videre.

Dette oppnår man med koden under.

Dere trenger ikke kunne dette, og vi kommer ikke til å bruke input noe videre i dette kurset.

                    <script>
                    
                    var count = 0;
                    function inputTest() {
                        var text = document.getElementById('inputText').value;
                        if (count < 3) {
                            outputTest(text);
                            count++;
                            if (count === 3){
                                document.getElementById('inputBtn').innerHTML = "Klikk én gang til";
                            }
                        }
                        else {
                            count = 0;
                            document.getElementById('inputBtn').innerHTML = "Trykk på meg";
                            document.getElementById('inputUl').innerHTML = "";
                        }
                    }
                    function outputTest(text) {
                        var ul = document.getElementById('inputUl');
                        var outputLi = document.createElement('li');
                        var outputText = document.createTextNode(text);
                        outputLi.appendChild(outputText);
                        ul.appendChild(outputLi);
                    }
                    </script>

                <button id="inputBtn" onclick="inputTest()">Trykk på meg</button>
                <input id="inputText" type="text">
                <ul id="inputUl">
                </ul>

For å enkelt kunne vise frem produktene med minst mulig HTML-kode skal vi benytte oss av Bootstrap-klasser som bruker for å vise lister på en nettside. Du kan starte med å lage en ul-tag inne i div-taggen med container-klassen. 'ul' står for 'Unordered list'. Denne nye ul-taggen skal ha følgende attributt: class="list-group".

Inne i din nye ul-tag, kan du lage en ny li-tag. 'li' er et element i en liste. Denne skal ha list-group-item som class. En li med list-group-item vil da representere et produkt.

Oppgave:Når du er ferdig med dette skal du legge til bilde og tekst av produktet, og en knapp med teksten 'Legg i handlekurv' på. Om du sitter fast kan du trykke på knappen under og se et løsningsforslag.

        <ul class="list-group">
            <li class="list-group-item">
                <img class="produkt" src="../img/mobil.png">
                Mobil
                <button class="btn btn-success">
                    Legg til
                </button>
            </li>
        </ul>

Legg også til flere produkter, slik at du totalt har 3 produkter tilgjengelig.

4. Legge til produkt i handleliste

Vi har nå laget all HTML og CSS for første versjon av webshoppen vår. Nå skal vi videre legge til funksjonalitet for å legge til produkter i en 'handleliste'. For å gjøre dette må vi bruke Javascript.

Det første vi kan gjøre er å skrive følgende inn i Javascript filen vår (script.js): var items = [];

Her har vi laget en variabel som vi har kalt for 'items'. Denne har fått verdien '[ ]', som er en liste/tabell. I denne listen kan vi legge til forskjellige type verdier. I vårt tilfelle skal vi legge inn hvilke produkter kunden har valgt.

Vårt neste steg er å faktisk legge produkter inn i lista vi har deklarert (opprettet). Her må vi opprette en funksjon. En funksjon inneholder en eller flere kodelinjer som kan kjøres på et bestemt tidspunkt. Følgende skrives i scripts.js: 

function addItem(item) {
    items.push(item);
}

Her har vi laget en funksjon som heter 'addItem', som tar i mot et produkt og legger dette inn i lista vår.

Videre skal vi si i html-filen vår at denne skal kalles når vi trykker på knappen til et produkt. Vi legger inn følgende attributt i vår button-tag tilhørende et produkt: onclick="addItem('produktNavn');" (Her må du skrive selve navnet på produktet ditt og ikke produktNavn. Legg også merke til forskjell på enkeltsnutter (') og dobbeltsnutter ("))

For å teste at dette funker kan vi legge til en ny funksjon:

function seeItems() {
    if (items.length === 0) {
        alert('Du har ikke valgt noe!');
    }else{
        alert('Du har valgt: ' + items);
    }
}

Og for å kalle på denne funksjonen legger vi til følgende attributt i knappen hvor det står 'Se handlekurv': onclick="seeItems();"

Hvis du nå har gjort alt korrekt, så kan du forsøke å legge til et produkt i handlekurven også trykke på 'Se handlekurv', så skal du kunne se produktet du valgte komme opp som en varsel.

Oppgave: Legg til funksjonalitet for knappen 'Tøm handlekurv'.
